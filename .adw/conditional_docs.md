## Conditions

- path:README.md — when working on project setup, onboarding, or deployment
- path:next.config.ts — when modifying Next.js configuration, redirects, rewrites, or build settings
- path:tsconfig.json — when changing TypeScript compiler options or path aliases
- path:eslint.config.mjs — when updating linting rules or adding ESLint plugins
- path:postcss.config.mjs — when modifying Tailwind CSS or PostCSS configuration
- path:src/app/ — when working on any page, route, or app-level layout
- path:src/components/ — when working on shared UI components
- path:public/ — when managing static assets
